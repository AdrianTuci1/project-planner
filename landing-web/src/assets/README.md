# Assets
Place your images in this folder.
Recommended naming convention:
- `hero-visual.png`
- `feature-1.png`, `feature-2.png`, etc.
- `secondary-hero.png`

You can then import them in the respective components:
```jsx
import heroImage from '../assets/hero-visual.png'
// ...
<img src={heroImage} alt="Hero" />
```
